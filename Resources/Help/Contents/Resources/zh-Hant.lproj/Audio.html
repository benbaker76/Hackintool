音訊修補
  1. 安裝最新的 Lilu + AppleALC
  2. 通過 Clover Configurator 設定 config.plist/Devices/Audio/Inject=NO
  3. Clover->DSDT 重命名 (如果偵測到)
    • AZAL->HDEF
    • HDAS -> HDEF
  4. 在 Hackintool 工具中點擊 聲音
  5. 通過下拉列表設定 Layout ID
  6. 文件->匯出->Clover config.plist (覆蓋現有的config.plist文件)
  7.  重新啟動
  8. 如果沒有聲音，請返回步驟4並嚐試不同的 Layout ID

注意：如果您嚐試所有 Layout ID 但仍有問題，請嚐試勾選 內顯-> 執行修補-> 進階-> 仿冒音效卡ID 選項。 如果這依然不起作用，那麼嚐試安裝 FakePCIID.kext + FakePCIID_Intel_HDMI_Audio.kext
